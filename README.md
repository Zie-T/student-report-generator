# Student Report Generator

A Python command-line tool that generates student report cards.

## Features
- Accepts user input for student names and marks
- Validates input and handles errors gracefully
- Calculates class average
- Identifies top student
- Shows pass/fail for each student
- Saves report to a .txt file

## How to Run
python studentreport_generator.py
